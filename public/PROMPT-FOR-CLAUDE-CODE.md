## ПРОМПТ (копировать отсюда):

Собери мобильную версию главной страницы интернет-магазина кроссовок.
Используй только HTML + CSS + минимальный JS. Без фреймворков.
Файл — один index.html.

### Дизайн-система

```
Шрифт: Manrope (Google Fonts), веса: 400, 500, 600, 700, 800
Фон страницы: #FFFFFF
Основной текст: #1A1A1A
Второстепенный текст: #888888
Границы и разделители: #F0F0F0
Фон карточек/плейсхолдеров: #F5F5F5
Акцентный (лайк): #E53935
Telegram кнопка: #2AABEE

Скругления:
- Маленькие элементы (бейджи, теги): 10px
- Карточки товаров, преимущества: 14px
- Категории: 20px  
- Баннеры: 24px
- Кнопки: 50px (полностью скруглённые)

Ширина контейнера: max-width: 430px, margin: 0 auto
Боковые отступы контента: 16px
```

### Структура страницы (сверху вниз)

**1. HEADER (sticky)**
- Слева: иконка гамбургер-меню (три полоски)
- По центру: текст "Лого" (bold, uppercase, letter-spacing: 1px)
- Справа: 3 иконки — поиск (лупа), избранное (сердце), корзина (сумка)
- Фон: белый полупрозрачный с backdrop-filter: blur(12px)
- Нижняя граница: 1px solid #F0F0F0

**2. HERO-БАННЕР**
- Отступы: margin 12px 16px
- Скругление: 24px
- Высота: ~200px
- Фон: градиент тёплых бежевых оттенков (имитация фото кроссовок)
- Контент в левом нижнем углу:
  - Заголовок: "ВАШИ ЛЮБИМЫЕ ВЕСЕННИЕ МОДЕЛИ", font-size 22px, font-weight 800, uppercase, max-width 240px
  - Кнопка: "В каталог", белый фон, скругление 50px, padding 10px 20px, font-weight 600

**3. КАТЕГОРИИ**
- Два блока в ряд (grid 1fr 1fr, gap 10px)
- Padding: 0 16px
- Каждый блок: скругление 20px, высота 130px
- Левый "Мужчинам": бежево-серый градиент
- Правый "Женщинам": розовато-серый градиент
- Текст категории: position absolute, bottom 12px, left 14px, font-weight 700

**4. ХИТЫ СЕЗОНА**
- Заголовок: "Хиты сезона", 22px, font-weight 800, padding 28px 16px 0
- Табы брендов (горизонтальный скролл): Balenciaga | Adidas | Nike | Golden Goose
  - Активный таб: цвет #1A1A1A, подчёркивание 2px solid снизу
  - Неактивный: цвет #888888
- Карусель товаров (горизонтальный скролл, snap):
  - Ширина карточки: calc(50% - 5px)
  - Изображение: фон #F5F5F5, скругление 14px, высота 180px
  - Кнопка-сердце: position absolute top-right, белый круг 32px, тень
  - Бейдж скидки: "-20%", position absolute top-left
  - Название товара: 13px, font-weight 500, max 2 строки (line-clamp)
  - Цена: 15px, bold. Старая цена: 12px, зачёркнутая, серая
- Пагинация точками: 4 точки, активная — вытянутая (20px ширина), тёмная
- Кнопка "Смотреть все": на всю ширину, outline стиль, border 1.5px solid #F0F0F0, скругление 50px

**5. ВТОРИЧНЫЙ БАННЕР**
- margin: 24px 16px, скругление 24px, высота 200px
- Серо-бежевый градиент
- Кнопка "В каталог" в левом нижнем углу

**6. БРЕНДЫ**
- Заголовок "Бренды": 22px bold
- Горизонтальный скролл логотипов:
  - Каждый: 72x72px, border 1.5px solid #F0F0F0, скругление 14px
  - Бренды: NB, Adidas, Asics, PUMA, Reebok
- Кнопка "Все бренды": outline стиль, как "Смотреть все"

**7. НАШИ ПРЕИМУЩЕСТВА**
- 3 карточки вертикально, gap 8px
- Каждая: border 1.5px solid #F0F0F0, скругление 14px, padding 16px
- Содержание: иконка (44x44) | текст | стрелка вправо (шеврон)
- Пункты: Бесплатная доставка (иконка грузовик), Оригинальная продукция (галочка в круге), Оплата после примерки (иконка карты)

**8. CTA-БАННЕР**
- margin: 28px 16px, скругление 24px, padding 32px 24px
- Градиент: linear-gradient(135deg, #1B4B6B → #1A6B5A → #2A8B6A)
- Текст по центру: "ОСТАЛИСЬ ВОПРОСЫ?" (20px bold white) + "ЗАДАЙТЕ ИХ В ЧАТ ПОДДЕРЖКИ" (16px)
- Кнопка: "Наш Telegram", фон #2AABEE, белый текст, скругление 50px, с SVG-иконкой телеграма

**9. FOOTER**
- Фон: #F5F5F5, padding 28px 16px 40px
- Три секции со ссылками:
  - "Всё о заказе": Заказ и оплата, Доставка, Правила возврата
  - "Сервис и помощь": Часто задаваемые вопросы, Таблицы размеров, Контакты
  - "Юридический раздел": Персональные данные, Публичная оферта
- Заголовки секций: 15px, font-weight 800
- Ссылки: 13px, цвет #888888

### Требования к коду
- Mobile-first, max-width 430px
- Все скроллы горизонтальные — без видимого scrollbar (scrollbar-width: none)
- Иконки — inline SVG, stroke стиль
- Все кнопки — transition и :active эффект (scale 0.96)
- Header sticky с blur-эффектом
- Плавная анимация появления при загрузке (fadeUp на hero и категории)

## СОВЕТЫ ПО РАБОТЕ С CLAUDE CODE

1. **Начни с этого промпта целиком** — он задаст дизайн-систему
2. **После первой генерации** — проверь и корректируй посекционно: "Измени header — лого должно быть по центру, сейчас оно смещено влево"
3. **Если что-то не так** — описывай конкретно: "уменьши отступ между карточками товаров с 16px до 10px" вместо "поправь карточки"
4. **Для картинок** — используй временные placeholder-фоны, потом заменишь на реальные img
5. **Не проси "сделай красивее"** — проси "добавь тень 0 2px 8px rgba(0,0,0,0.1) на кнопку"
